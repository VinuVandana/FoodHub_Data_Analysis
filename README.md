# FoodHub Data Analysis
Exploratory data analysis of FoodHub customer orders using Python to identify customer behavior, restaurant performance, delivery efficiency, and business recommendations.

## Project Overview

This project was completed as part of the **Post Graduate Program in Artificial Intelligence** offered by **The University of Texas at Austin**.

The objective of this project is to analyze customer order data from FoodHub, an online food delivery platform, to uncover customer ordering patterns, evaluate delivery performance, and generate actionable business recommendations. Using exploratory data analysis (EDA) and data visualization techniques, this project provides insights that can help improve operational efficiency and customer satisfaction.

---

## Business Context

FoodHub is an online food delivery platform that connects customers with multiple restaurants. As the business grows, understanding customer behavior, restaurant performance, delivery efficiency, and order characteristics becomes essential for improving customer experience and increasing revenue.

This project analyzes historical order data to identify trends, detect operational bottlenecks, and recommend strategies that support data-driven business decisions.

---

## Objectives

- Perform Exploratory Data Analysis (EDA)
- Understand customer ordering behavior
- Analyze restaurant performance
- Evaluate delivery times and operational efficiency
- Identify factors influencing customer ratings
- Generate business recommendations based on data insights

---

## Dataset

The dataset contains historical customer order information, including:

- Order ID
- Customer ID
- Restaurant Name
- Cuisine Type
- Cost of the Order
- Day of the Week
- Food Preparation Time
- Delivery Time
- Customer Rating

## Business Recommendations

- Optimize delivery operations during peak demand periods.
- Partner more closely with high-performing restaurants.
- Improve delivery efficiency for restaurants with longer delivery times.
- Develop targeted promotions for popular cuisines and customer segments.
- Use customer behavior insights to improve marketing and retention strategies.
